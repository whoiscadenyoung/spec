# Decisions: date field

Add a date field to the record frontmatter using format: YYYY-MM-DD (2026-03-24).

Update the record frontmatter with a `date` field right beneath `slug`.

- `templates/decisions/examples/*`: Update every example
- `templates/decisions/records/000-use-decision-records.md`: Update the first record
- `templates/decisions/records/record-template.md`: Update the template with {date} placeholder
- `templates/decisions/README.md`: Update README documentation to consider this field.

Do not add the date field to the decision-log.
